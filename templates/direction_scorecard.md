## Decision Rules

Go only if:

- No hard filter fails.
- Mainline clarity ≥ 4.
- Depth / problem tree ≥ 4.
- Evaluation substrate ≥ 4.
- 3-month entry ≥ 4.
- Value axis fit ≥ 4.

Hold if:

- Root problem is strong but one evidence gap remains.
- Missing evidence can be checked within one additional week.
- No hard filter fails irreversibly.

Kill if:

- It is only a direction label.
- It lacks public evaluation substrate.
- It cannot start within 3 months.
- It cannot form a natural problem tree.
- Its 4-paper arc is a template.
- It violates value axis.
- It depends on private data or large engineering.
